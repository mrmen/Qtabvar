QTabVar

QTabVar is a software written in C++ using Qt. The current goal is to allow user edit mathematics tabular using GUI. At this moment, QTabVar can export tabular as PostScript or LaTeX code. Both of them can be edited using the "code" tab.

LaTeX code

LaTeX code use tikzrput.sty file which can be obtained on altermundus website.

PostScript code

PostScript code use two files (skel.ps and skel-variation.ps) to generate output.
